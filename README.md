# About
Backend example for implement JWT with frontend vue

# Instalation
- clone this repo
```
git clone git@github.com:wildanjisung/example-be-crud-nestjs.git
```
- install package
```
npm install
```
- run this project
```
nest start -b swc -w
```
- run this project with debug
```
nest start -b swc -w --debug
```
